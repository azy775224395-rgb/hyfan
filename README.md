
# متجر أبو إيفان للطاقة المتجدّدة ☀️

موقع تجارة إلكترونية متخصص في حلول الطاقة الشمسية، البطاريات، والأجهزة المنزلية الموفرة للطاقة.

## المميزات:
- **نظام سلة مشتريات:** تجميع المنتجات وإرسال طلب كامل عبر واتساب.
- **طلب واتساب مباشر:** إرسال بيانات المنتج مع الرابط الخاص به وتلقي الطلب فوراً.
- **مساعد ذكي (AI):** مدعوم بـ Google Gemini للإجابة على استفسارات العملاء الفنية.
- **تصميم عصري:** متجاوب بالكامل مع الهواتف الذكية والحواسيب.

## 🔑 إعداد مفتاح الذكاء الاصطناعي (API Key)
لجعل "مساعد حيفان" يعمل، تحتاج إلى مفتاح Google Gemini:

1. احصل على المفتاح مجاناً من: [Google AI Studio](https://aistudio.google.com/app/apikey)
2. **في جهازك المحلي:**
   - قم بإنشاء ملف جديد باسم `.env` بجانب ملف `package.json`.
   - أضف السطر التالي داخله:
     ```
     API_KEY=ضع_مفتاحك_هنا
     ```
3. **عند الرفع على Render:**
   - اذهب إلى **Environment**.
   - أضف `API_KEY` كـ Key ومفتاحك كـ Value.

## طريقة التشغيل محلياً:
```bash
npm install
npm run dev
```

## طريقة النشر (Deployment):
1. ارفع الكود إلى GitHub.
2. اربط المستودع بمنصة **Render** أو **Vercel**.
3. **هام جداً:** لا تنسَ إضافة `API_KEY` في إعدادات البيئة (Environment Variables) للمنصة.
